justCheckers - README
=====================

About the Project
-----------------

*justCheckers* is an advanced, cross-platform, libre source checkers game. The project's aim is to make a game capable
of supporting:

- Skinning
- Network Games
- Computer Opponents
- Various Rules and Internationalizations
- Multiple Desktop and Mobile OS
